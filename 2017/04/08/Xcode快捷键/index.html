Xcode快捷键

这些你必须熟练应用的Xcode快捷键

工欲善其事,必先利其器。 作为一个iOS程序猿你可别告诉我你在使用Xcode的时候除了码代码用键盘，你可别说你会的快捷键只有 清理 编译 和 运行 😂，其他操作都使用鼠标完成哦。那么我们想想要是我们双手不离开键盘就可以完成大部分的事情是多么的拉风。今天就来给大家总结一下Xcode的使用快捷键，来方便大家提高编程效率。

快速浏览代码、图片界面文件

我擦我界面现在不知道跳哪了？ 别着急 使用这个快捷键使你的工程导航器显示你的代码文件。其实啊,上边一排的按钮 的快捷键都是 command + 对应的位置数字

command + 1

隐藏导航器

是不是有时候觉得自己编码的空间不够,上图只是夸张一下😄,不过有时候的确是偏小,在写代码的时候也可以关闭一些东西
  1. command + shift + Y 关闭/打开 底部的输出控制台
  2. command + 0 关闭/打开 左侧的导航器面板
  3. command + option + 0 关闭/打开 右侧的导航器面板

快速的全局搜索

有时候想在全局搜索一个东西,复制完了之后 要好几步才能完成 太太太繁琐了

command + shift + F 直接打开全局搜索 粘贴回车就可以看结果了

文件栏跳转

在当前文件代码已经上千行了,找一个方法还是在上下翻动？ 岂不是累死了。使用文件栏跳转快速的进行小范文的查找，并跳转到该方法的位置

control + 6

快速全局定位跳转到指定代码

当你想找一个方法的时候而你却不太记得到底是哪个文件里了，又不想一个一个点开去找那么这个全局定位跳转很适合你

command + shift + O

打开辅助编辑器窗口

当你在Xib视图需要关联文件了，或者你在编辑的时候需要打开一个辅助窗口别再傻傻的去右上角点击了
  1. command + option + enter 打开辅助窗口
  2. command + enter 关闭辅助窗口

更新自动布局

在Xib给控件设置好了约束之后直接快捷键一步更新位置

command + option + =

快速切换.h 和 .m 文件

在书写代码的时候可以快速的在.h .m 文件之间迅速的切换

command + option + (↑/↓)

快速移动光标和快速删除代码

快速的将光标移动到某一个位置，快速的删除某一行代码或者某一个单词
  1. command + delete 删除当前光标之后的本行代码
  2. option + delete 删除当前光标之后的一个单词
  3. command + ←/→ 移动到当前行的行首或者行尾
  4. option + ←/→ 移动光标到的前一个或者后一个单词
  5. command + ↑/↓ 将光标移动到文件的顶部/尾部

以上所有操作加上shift即可选中

关闭/展开代码块

command + option + ←/→
